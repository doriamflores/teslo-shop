# Teslo API

1. Clonar proyecto

2. `npm install`

3. Copiar el `.env.template` a `.env` y modificarlo.

4. Cambiar las variables de entorno.

5. Levantar la base de datos

```
docker-compose up -d
```

6. Ejecutar SEED

```
http://localhost:3000/api/seed
```

7. Ejecutar el proyecto

```
npm run start:dev
```
